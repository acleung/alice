To get started:

1. Install Android USB Driver: http://developer.android.com/sdk/win-usb.html
2. Open GGeneration, go to the "boss" quest. Use the "sort" function to make sure the boss quest you want is the top one.
3. Enter the boss, fix your team.
4. Leave the screen at the quest screen with that big red button in the middle.
5. (Windows) Go back to windows, click on ggenbot.exe
5. (Mac) Go to terminal, navigate to the directory by typing "cd /Users/name/Desktop/ggenbot/" then type "./ggenbot_mac"


To change action sequences:

Rename the battle_data_blah_blah.txt to battle_data.txt
